## Enable GitHub Pages

1. In this repo go to **Settings → Pages**.
2. Under **Source** choose **Deploy from a branch**.
3. Branch: **main** (or **master**), folder: **/ (root)** → **Save**.
4. Wait 1–2 minutes; your site will be at `https://<username>.github.io/<repo-name>/`.
